\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_request}{Request}} class simulates the requests of random requirements being sent to the \mbox{\hyperlink{class_load_balancer}{Load\+Balancer}}. 

This class does this by generating a in and out IP address, time to complete, and keeps track of the \mbox{\hyperlink{class_web_server}{Web\+Server}} that created it using web\+Server\+ID.
